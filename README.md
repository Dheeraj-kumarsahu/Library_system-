### 📚 Library Management Software

A proper, simple, and efficient Library Management Software designed to streamline day-to-day library operations for both books and students.

### 🎯 Purpose

The primary objective of this project is to provide a user-friendly digital system that automates record-keeping, eliminates manual paperwork, and tracks library assets efficiently.

### ✨ Features

### 📖 Book Management

*   **Store:** Add new books with titles, authors, and unique IDs.
*   **View:** Browse the complete catalog or search for specific books.
*   **Update:** Modify book details easily.
*   **Delete:** Remove outdated or lost books from the system.

### 🎓 Student Management

*   **Store:** Register new students into the library system.
*   **View:** Check student profiles and their borrowing history.
*   **Update:** Edit student contact information or status.
*   **Delete:** Remove student records upon graduation or departure.

### 💰 Core Operations

*   **Late Fee Calculation:** Automatically calculate and manage late fees for delayed book returns.

### 🛠️ Tech Stack

*   **Front-End:** Python (Provides the user interface and core application logic)
*   **Back-End:**
    *   **MySQL:** Handles structured data like student profiles and book inventories.
    *   **TXT Files:** Logs transactions, system backups, or quick-reference configurations.
